# 毕业设计创新点分析：基于手势识别的智能云台控制系统

## 项目概述

本项目实现了一个基于计算机视觉的手势识别系统与STM32嵌入式PID控制的闭环云台控制系统。PC端使用MediaPipe实时追踪手部21个关键点，结合自适应EMA滤波与速度预测减少视觉延迟，再通过模糊自适应PID（FuzzyPID）进行参数优化补偿；STM32接收坐标数据后解析协议，执行位置式PID控制驱动双轴伺服电机，实现手势驱动的精确跟踪。

### 创新点1: 自适应EMA滤波 + 速度预测控制

**解决问题**：MediaPipe追踪抖动与传输延迟导致的云台控制抖动

**理论**：
- EMA（指数移动平均）滤波：低通滤波器，抑制高频噪声
- 一阶预测：基于历史速度外推下一帧位置，补偿系统延迟

**数学公式**：

EMA滤波：
$$EMA_k = \alpha \cdot x_k + (1 - \alpha) \cdot EMA_{k-1}$$

速度自适应alpha：
$$\n\alpha = \begin{cases} \alpha_{fast} = 0.15 & \text{if } v > v_{th} \\ \alpha_{slow} = 0.05 & \text{if } v \leq v_{th} \end{cases}
$$

一阶预测：
$$\n\hat{x}_{k+1} = EMA_k + (EMA_k - EMA_{k-1})
$$

### 创新点2: 模糊推理 + ANFIS在线学习混合架构

**解决问题**：云台负载变化、外部扰动导致固定PID振荡/超调

**理论**：
- 模糊逻辑控制（Mamdani推理）：基于专家经验的规则库
- ANFIS（自适应神经模糊推理系统）：在线学习模糊规则输出
- 混合架构：模糊查表作为"教师信号"引导ANFIS学习

**数学公式**：

三角隶属函数：
$$
\mu_{triangle}(x; a,b,c) = \max\left(\min\left(\frac{x-a}{b-a}, \frac{c-x}{c-b}\right), 0\right)
$$

模糊规则表（3×3 = 9规则）：
| e/de | N (负) | Z (零) | P (正) |
|------|--------|--------|--------|
| N    | (P,P)  | (P,Z)  | (Z,P)  |
| Z    | (Z,P)  | (Z,Z)  | (Z,P)  |
| P    | (P,P)  | (P,Z)  | (Z,P)  |

输出反模糊化：
$$
\Delta K_p = \frac{\sum \mu_i \cdot K_{p,i}}{\sum \mu_i}
$$

ANFIS前向传播（9规则高斯隶属）：
$$
w_i = \exp\left(-\frac{1}{2}\left(\frac{e-c_i}{\sigma_i}\right)^2\right) \cdot \exp\left(-\frac{1}{2}\left(\frac{de-c'_i}{\sigma'_i}\right)^2\right)
$$

参数更新（反向传播 + L2正则）：
$$
\Delta p = -lr \cdot (w \cdot err \cdot e + 2\lambda \cdot p)
$$

**代码片段** (`FuzzyPID.py:75-86`, `108-117`)：

```python
def update(self, e, de, dt=0.033):
    # 1. 模糊查表作为教师信号
    dkp_fuzzy, dkd_fuzzy = self.infer_fast(e, de)

    # 2. ANFIS前向计算
    dkp, dkd, w_bar = self.anfis.forward(e, de)

    # 3. ANFIS反向学习（模仿模糊）
    loss = self.anfis.backward(e, de, dkp_fuzzy, dkd_fuzzy, w_bar)


    # 4. 自适应参数
    self.Kp = np.clip(self.Kp_base + dkp, 0.01, 0.3)
    self.Kd = np.clip(self.Kd_base + dkd, 0.05, 0.5)

    # 5. 控制量计算（带积分限幅 + 输出平滑）
    self.integral_e = np.clip(self.integral_e + e * dt, -100, 100)
    u_raw = self.Kp * e + self.Ki_base * self.integral_e + self.Kd * de
    self.u_smooth = 0.8 * self.u_smooth + 0.2 * u_raw
    return self.u_smooth, self.Kp, self.Kd, loss
```

```python
def forward(self, e, de):
    # 高斯隶属度
    mu_e = np.array([self.gaussian_memb(e, self.c_e[i], self.sigma_e[i]) for i in range(3)])
    mu_de = np.array([self.gaussian_memb(de, self.c_de[i], self.sigma_de[i]) for i in range(3)])
    w = np.outer(mu_e, mu_de).flatten()[:9]

    # 归一化 firing strength
    w_bar = w / (np.sum(w) + 1e-8)


    # 线性函数输出
    f_kp = np.clip(self.p_kp*e + self.q_kp*de + self.r_kp, -0.05, 0.05)
    f_kd = np.clip(self.p_kd*e + self.q_kd*de + self.r_kd, -0.1, 0.1)


    # 加权求和
    dkp = np.sum(w_bar[:,np.newaxis] * f_kp)
    dkd = np.sum(w_bar[:,np.newaxis] * f_kd)
    return dkp, dkd, w_bar
```


### 创新点4: 基于SVM的归一化特征手势分类与三态状态机

**解决问题**：手势位置、大小变化导致识别精度下降

**理论**：
- 支持向量机（SVM）：线性核，最大化类间间隔
$$f(x) = \text{sign}(w \cdot x + b)$$
- **特征归一化**：将绝对像素坐标归一化到[0,1]范围，消除位置和尺度敏感性
- 状态机：有限状态自动机防止误触发

**问题分析**：
- 旧方案使用绝对像素坐标 `(x, y)`，手在图像左侧和右侧特征值差异大
- 手靠近或远离摄像头时，绝对坐标数值范围变化大
- 归一化后特征范围固定为 `[0, 1]`，无论位置和距离如何变化

**数学公式**：

坐标归一化：
$$x_{norm} = \frac{x}{w}, \quad y_{norm} = \frac{y}{h}$$
其中 `w` 为图像宽度，`h` 为图像高度。

SVM决策边界：
$$\frac{2}{||w||} \text{ 最大间隔}$$

Pipeline标准化：
$$x' = \frac{x - \mu}{\sigma}$$

### 创新点5: PC-STM32自定义串口协议与闭环集成

**解决问题**：跨平台实时通信丢包、同步

**协议格式**：`#x$y\r\n`
- `#`：坐标起始标记
- `x`：X轴坐标（像素值）
- `$`：分隔符
- `y`：Y轴坐标
- `\r\n`：结束标记

**STM32解析** (`serial.c:31-58`)：


**STM32位置式PID** (`pid.c:16-38`)：

```c
int pid(int present, u16 target, PID_TypeDef* PID) {
    PID->error = target - present;  // 误差 = 目标 - 实际

    PID->p_out = PID->Kp * PID->error;               // 比例
    PID->i_out += PID->Ki * PID->error;             // 积分
    if (PID->i_out > 500) PID->i_out = 500;         // 积分限幅
    if (PID->i_out < -500) PID->i_out = -500;

    PID->d_out = PID->Kd * (PID->error - PID->last_error);  // 微分

    PID->output = PID->p_out + PID->i_out + PID->d_out;
    if (PID->output > 1000) PID->output = 1000;    // 输出限幅
    if (PID->output < -1000) PID->output = -1000;

    PID->last_error = PID->error;
    return PID->output;
}
```

## 总结

本系统将计算机视觉、模式识别、自适应控制有机结合，实现了手势驱动的智能云台控制。核心创新在于：

1. **速度自适应EMA滤波 + 一阶预测**：有效抑制MediaPipe追踪抖动，预测补偿延迟
2. **模糊-ANFIS混合架构**：结合专家经验的可解释性与在线学习自适应性
3. **级联控制**：PC端FuzzyPID预处理 + STM32位置式PID执行的双层架构
4. **归一化特征SVM分类 + 三态防误触状态机**：位置/尺度不变的手势识别，确保命令可靠性
5. **自定义串口协议**：简洁高效，易于嵌入式实现

系统适用于无人机云台、智能机器人无接触控制等场景。
